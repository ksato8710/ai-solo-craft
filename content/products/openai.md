## OpenAIとは？

GPT-4o、GPT-4 Turbo、GPT-3.5など複数モデル。ステートフルなAIアシスタントを構築。画像生成・音声認識APIも提供。

## 主要機能

### GPTシリーズ
GPT-4o、GPT-4 Turbo、GPT-3.5など複数モデル。

### Assistants API
ステートフルなAIアシスタントを構築。

### DALL-E / Whisper
画像生成・音声認識APIも提供。

## 料金

API従量課金（GPT-4o: $5/1M input tokens）

## ソロビルダー向けの使いどころ

- プロダクト開発の効率化
- 反復作業の自動化
- 品質向上と時間短縮

## 注意点・制限

- 従量課金で大量利用時はコストが膨らむ
- Rate limitの管理が必要
- モデル更新で挙動が変わることがある
