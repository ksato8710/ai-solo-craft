## Kimi Codeとは？

Kimi Codeは、[Kimi K2.5](/products/kimi-k25)を搭載した**オープンソースのエージェンティックコーディングツール**。2026年2月にリリースされ、AIツールランキングで初登場4位を獲得（LogRocket調査）。

**ターミナルファースト**の設計で、CLIから直接AIエージェントを呼び出してコード生成・編集を実行。VSCode、Cursor、Zedとの統合機能も備え、IDEからシームレスに利用可能。

最大の特徴は**画像・動画入力によるビジュアルデバッグ**。エラー画面のスクリーンショットやUI動作の録画を渡すだけで、問題を自動診断・修正。また、既存のMCPサーバーやスキル設定を**自動検出・移行**する機能があり、Claude Codeなど他ツールからの乗り換えがスムーズ。

## 主要機能

### ターミナル統合

CLIから直接AIエージェントを呼び出し。`kimi "この関数をリファクタリングして"` のようにシンプルなコマンドでコード編集を実行。パイプやリダイレクトと組み合わせて柔軟なワークフローを構築。

### IDE統合（VSCode / Cursor / Zed）

VSCode、Cursor、Zedに拡張機能としてインストール。エディタ内からKimi K2.5のパワーを活用。既存のIDE設定を活かしながら、高度なAIコーディング支援を追加。

### 画像・動画入力（ビジュアルデバッグ）

エラー画面のスクリーンショット、UIバグの動画を入力として渡すと、AIが視覚的に問題を分析。「この画面でボタンが反応しない」といった曖昧な報告から、具体的な修正コードを生成。

### Agent Swarm連携

Kimi K2.5のAgent Swarm機能を活用。最大100のサブエージェントで複雑なタスクを並列処理。大規模リファクタリングや複数ファイル同時編集に威力を発揮。

### MCP・スキル自動移行

Claude Code、Cursor、Aiderなど他ツールで設定したMCPサーバーやカスタムスキルを自動検出。設定ファイルを移行する手間なく、すぐに既存のワークフローを継続。

### オープンソース

Modified MITライセンスで完全オープンソース。コードを自由にカスタマイズ、自社環境に最適化、社内ツールに組み込みが可能。

## 使い方（Getting Started）

### CLIインストール

```bash
# npm
npm install -g @moonshot/kimi-code

# または直接ダウンロード
curl -fsSL https://kimi.moonshot.cn/install.sh | sh
```

### IDE拡張インストール

- **VSCode**: 拡張機能で「Kimi Code」を検索
- **Cursor**: 同様に拡張機能からインストール
- **Zed**: プラグインマネージャーから導入

### 基本コマンド

```bash
# コード生成
kimi "Reactでログインフォームを作成"

# ファイル編集
kimi edit src/App.tsx "エラーハンドリングを追加"

# 画像からデバッグ
kimi debug --image error-screenshot.png

# 動画からUI分析
kimi analyze --video ui-bug.mp4

# MCP移行
kimi migrate --from claude-code
```

## FAQ

### Q: Claude Codeとの違いは？

A: Kimi Codeはオープンソースで無料。ビジュアルデバッグ（画像・動画入力）とAgent Swarm（並列処理）が独自機能。Claude Codeは安定性とAnthropic統合が強み。

### Q: Aiderとの違いは？

A: AiderはGit統合が強み。Kimi CodeはAgent Swarmによる並列処理、ビジュアル入力、IDE統合が特徴。両方使い分けるユーザーも多い。

### Q: ローカルで動かせる？

A: はい。Kimi K2.5をセルフホストすれば完全ローカル動作可能。VRAM 24GB以上推奨。

### Q: 既存のMCP設定は使える？

A: はい。`kimi migrate --from claude-code` で既存設定を自動検出・移行。手動設定も可能。

### Q: 日本語対応は？

A: 対応しているが、英語/中国語が最も精度が高い。技術的な指示は英語推奨。

## ソロビルダー向けの使いどころ

### ターミナルワークフロー

シェルスクリプトやMakefileにKimi Codeを組み込み、ビルドパイプラインにAI処理を統合。CI/CDでのコード品質チェックを自動化。

### ビジュアルバグ修正

QAチームやユーザーから送られたエラースクリーンショットをそのまま入力。視覚的な情報から問題を特定し、修正コードを生成。

### 大規模移行

レガシーコードのモダン化をAgent Swarmで並列実行。数百ファイルのリファクタリングを一括処理。

### コスト最適化

Claude Code / Cursorの代替として導入。オープンソース + セルフホストでAPI費用を大幅削減。
