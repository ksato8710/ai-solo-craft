## 概要

![GitHub Copilot in Zed — 設定画面](https://github.blog/wp-content/uploads/2026/02/552265254-118225e1-4eb3-4491-9e8f-c77601b9f31c.jpg)
*出典: [GitHub Blog](https://github.blog/changelog/2026-02-19-github-copilot-support-in-zed-generally-available/)*

GitHubが2月19日、Zedエディタへの公式Copilotサポートを発表しました。これにより、Copilot Pro、Pro+、Business、Enterpriseの全サブスクリプションで、追加ライセンスなしにZedからCopilotを利用できるようになりました。

**出典:** [GitHub Blog](https://github.blog/changelog/2026-02-19-github-copilot-support-in-zed-generally-available/) — 2026-02-19

## 詳細

### Zedとは

Zedは、AtomとTree-sitterの開発者が作った高性能コードエディタです。Rust製で、以下の特徴があります:

- **超高速**: ネイティブ性能、大規模ファイルも軽快に動作
- **マルチプレイヤー**: リアルタイム共同編集に対応
- **モダンUI**: 洗練されたインターフェース

VS CodeやCursorが「AI-first」を掲げる中、Zedは「Speed-first」のアプローチを取ってきました。今回のCopilot統合により、その高速性を維持しながらAI支援も得られるようになります。

### 設定方法

ZedでCopilotを使い始める手順:

1. Zedの **Settings** を開く
2. **GitHub Copilot Chat** セクションに移動
3. **Sign in to use GitHub Copilot** をクリック
4. 認証フローを完了

これだけで、既存のCopilotサブスクリプションがZedで有効になります。

### 対象プラン

| プラン | Zedでの利用 |
|--------|------------|
| Copilot Pro | ✅ |
| Copilot Pro+ | ✅ |
| Copilot Business | ✅ |
| Copilot Enterprise | ✅ |

追加費用は一切不要です。

### 他エディタとの比較

| エディタ | 特徴 | Copilot対応 |
|----------|------|------------|
| VS Code | 拡張性、普及率 | ✅ ネイティブ |
| Cursor | AI-first設計 | ✅ + 独自AI |
| Zed | 高速性、共同編集 | ✅ 公式サポート |
| Neovim | カスタマイズ性 | ✅ プラグイン |

## ソロビルダーへの示唆

ZedへのCopilot統合は、以下のようなソロ開発者に特に有用です:

1. **VS Code重すぎ派**: 大規模プロジェクトでエディタの重さに悩む開発者が、Zedの軽量さとCopilotの便利さを両立できる
2. **Cursor以外の選択肢を探している人**: AI機能は欲しいが、VS Code派生から離れたい開発者向け
3. **チームでの共同編集**: Zedのマルチプレイヤー機能とCopilotを組み合わせたペアプロが可能
4. **既存Copilotユーザー**: 追加コストなしでエディタの選択肢が増えた

特に、「Copilotは使いたいが、VS Codeの動作が重い」という不満を持つ開発者にとって、Zedは有力な選択肢となります。
