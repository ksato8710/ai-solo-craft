## 概要

Anthropicは2026年2月、**年間売上ラン率が$14B**（約2.1兆円）に達したことを明らかにした。過去3年連続で年10倍以上の成長を記録しており、ソフトウェア企業として史上最速の成長軌道にある。

特に注目すべきは**Claude Code**の急成長で、年間ラン率は$2.5B超。2026年1月1日からわずか6週間で倍増した。

**出典:** [Economic Times](https://economictimes.indiatimes.com/news/international/us/anthropic-hits-14-billion-revenue-run-rate-becomes-fastest-growing-software-company-ever/articleshow/128304436.cms) — 2026-02-13

## 詳細

### 成長の実態

| 指標 | 数値 |
|------|------|
| 年間売上ラン率 | $14B（約2.1兆円） |
| Claude Code ラン率 | $2.5B超 |
| 年間$100K以上顧客 | 前年比7倍 |
| 年間$1M以上顧客 | 500社以上 |
| Fortune 10のうち | 8社が利用 |

### 資金調達

先日発表された**Series Gで$30B調達、$380B評価額**も、この成長を裏付けている。GIC、Coatue、D.E. Shaw Ventures、Dragoneer、Founders Fund、ICONIQなどが出資。

### 競合との比較

OpenAIは2024年末時点で年間売上$20B（ラン率）と報告している。Anthropicの$14Bは急速に差を詰めており、特にClaude Code vs GPT-5.3 Codexの開発者ツール競争が激化している。

## ソロビルダーへの示唆

### Claude Codeの信頼性

$2.5Bの売上は、多くの開発者・企業がClaude Codeに依存していることの証明だ。これは:

- **APIの安定性向上**: 売上があればインフラ投資も継続
- **機能拡充の継続**: 開発リソースの確保
- **長期的なサービス継続性**: 事業として成立している証拠

### 選択肢としての位置づけ

OpenAI（Codex）、GitHub（Copilot）、Anthropic（Claude Code）の三つ巴の競争は、開発者にとって選択肢の増加を意味する。

- 特定ベンダーロックインを避けられる
- 価格競争による恩恵の可能性
- 各ツールの強みを使い分けられる

Anthropicの急成長は、AIコーディング支援市場が本格的に立ち上がっていることの指標でもある。
