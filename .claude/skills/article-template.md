# Article Template — 記事テンプレートスキル

## 概要
カテゴリ別の記事構造テンプレートと、frontmatter仕様。

## Frontmatter（全記事共通）

```yaml
---
title: "記事タイトル"
slug: "url-friendly-slug"
date: "YYYY-MM-DD"
category: "morning-news"
description: "記事の要約（120文字以内）"
readTime: 5
featured: false
image: "/images/xxx.jpg"
---
```

## カテゴリ別テンプレート

### 🌅 朝のAIニュース（morning-news）

```markdown
# [日付] 朝のAIニュース — [メインテーマ]

## 🔥 ヘッドライン

### [メインニュースのタイトル]
[詳細な解説 — 定量データ、背景、ソロビルダーへの影響]

**出典:** [リンク]

## 📋 ニュースブリーフ

### [ニュース2のタイトル]
[簡潔な要約（3-5行）]

### [ニュース3のタイトル]
[簡潔な要約（3-5行）]

## 💡 ソロビルダー向け注目ポイント
[今日のニュースからソロビルダーが知っておくべきこと]
```

### 🛠️ 注目ツール（featured-tools）

```markdown
# [ツール名]、[何が起きたか] — [意味]

## 概要
| 項目 | 内容 |
|------|------|
| サービス名 | [名前] |
| 開始時期 | [YYYY年MM月] |
| 運営 | [企業名/個人名] |
| 調達額 | [$XX / 非公開] |
| MAU | [数値 / 推定値] |
| GitHub Stars | [数値（該当する場合）] |

## なぜ今注目されているか
[コミュニティの反応、直近のイベント、数字の変化]

## 機能と料金
[主要機能の説明 + 料金体系 + 無料枠の有無]

## ソロビルダー視点
[個人で使えるか？収益化に繋がるか？技術的な注意点]

## 競合との比較
[主要な競合ツールとの差分]

## ニュースバリュー評価
[NVAスコア表 + 所見 + 🗣 生の声（日本語）]

**出典:** [リンク]
```

### 🔬 深掘り・ハウツー（deep-dive）

```markdown
# [タイトル — 読者が何を得るか]

## この記事で分かること
- [ポイント1]
- [ポイント2]
- [ポイント3]

## [セクション1: 背景・前提]
[テーマの背景説明]

## [セクション2: 本論]
[詳細な解説、手順、分析]

## [セクション3: 実践]
[ソロビルダーが今すぐ活かせる具体的なアクション]

## まとめ
[要点の整理 + 次のステップ]

**参考リンク:** [出典一覧]
```

### 📊 事例分析（case-study）

```markdown
# [ビルダー名/プロダクト名] — [成果の要約]

## プロフィール
| 項目 | 内容 |
|------|------|
| ビルダー | [名前] |
| プロダクト | [名前] |
| 開始 | [時期] |
| 現在のMRR | [$XX] |
| ユーザー数 | [数値] |
| 技術スタック | [使用ツール] |

## ストーリー
[どうやって始めたか、どう成長したか]

## 収益モデル
[どうやって稼いでいるか、料金体系]

## 成功要因の分析
[なぜうまくいったか、再現可能性]

## ソロビルダーへの教訓
[この事例から学べること、注意すべきこと]

**出典:** [リンク]
```
