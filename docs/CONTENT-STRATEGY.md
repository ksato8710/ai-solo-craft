# AI Solo Builder — コンテンツ戦略

*作成日: 2026-02-03*

---

## コンテンツの目的

**「毎日まずこのサイトを見よう」と思わせる情報の網羅性と鮮度を確保する。**

---

## カテゴリ別コンテンツ設計

### 🌅 朝のAIニュース（morning-news）

- **配信:** 毎朝 8:00
- **内容:** 直近24時間のグローバルAIニュースから厳選キュレーション
- **構成:**
  - ヘッドライン（最重要ニュース 1-2本を詳しく）
  - ニュースブリーフ（3-5本を簡潔に）
  - ソロビルダー向け注目ポイント
- **情報ソース:** X, Reddit, Hacker News, Product Hunt, TechCrunch, The Verge
- **選定基準:** NVAスコア Tier A/B を優先

### 🛠️ 注目ツール（featured-tools）

- **配信:** 毎日 12:00
- **内容:** 注目AIプロダクト・ツールの深掘り紹介
- **構成:**
  - プロダクト概要（開始時期・運営者・資金調達）
  - 定量データ（MAU・ARR・GitHub Stars・ユーザー数）
  - 機能と料金体系（無料枠の有無）
  - ソロビルダー視点での活用方法
  - 競合との比較
  - NVA評価セクション（スコア表+所見+生の声）
- **選定基準:** コミュニティの盛り上がり + ソロビルダーとの関連度

### 🔬 深掘り・ハウツー（deep-dive）

- **配信:** 毎夜 19:00
- **内容:** AI開発プラクティス・戦略の詳細解説
- **テーマ例:**
  - 特定技術の詳細ガイド（Claude Code活用法、プロンプトエンジニアリング）
  - 収益化戦略の分析（どうやって稼いでいるか）
  - ツール比較・選定ガイド
  - AI業界のトレンド分析
- **選定基準:** 読者の実務に直結する実用性

### 📊 事例分析（case-study）

- **配信:** 不定期（良質な事例が見つかった時）
- **内容:** 成功/失敗事例の詳細分析
- **構成:**
  - ビルダーのプロフィール・背景
  - 使用ツール・技術スタック
  - 収益モデル・成長過程
  - 定量データ（MRR推移・ユーザー数推移）
  - 成功/失敗の要因分析
  - ソロビルダーへの教訓

## 記事品質基準

### 必須要件（全記事共通）

- [ ] タイトルに「何が起きたか」or「何を得るか」が含まれている
- [ ] 定量データ（MAU/調達額/Stars等）が最低1つ含まれている
- [ ] サービス紹介時に開始時期が明記されている
- [ ] 出典リンクが明記されている
- [ ] 「なぜ今盛り上がっているか」の文脈がある
- [ ] NVA評価セクション（featured-tools, morning-newsの主要記事）
- [ ] frontmatter が正しい（title, slug, date, category, description, readTime）

### 海外記事の紹介ルール

- **形式:** 要点紹介 + 日本市場への独自分析
- **禁止:** 全文翻訳の転載
- **必須:** 出典リンク、原文へのリスペクト
- **付加価値:** ソロビルダー視点での実現可能性・注意点
- **表記:** 「本記事は翻訳ではなく、原文の紹介・解説記事です」

## 配信スケジュール

| 時刻 | カテゴリ | 自動/手動 |
|------|---------|----------|
| 08:00 | 🌅 朝のAIニュース | cron自動 |
| 12:00 | 🛠️ 注目ツール | cron自動 |
| 19:00 | 🔬 深掘り/ハウツー | cron自動 |
| 不定期 | 📊 事例分析 | 手動 |

---

## 内部リンク戦略

- morning-news → 関連する featured-tools 記事へリンク
- featured-tools → 同カテゴリの deep-dive/case-study へリンク
- deep-dive → 関連ツールの featured-tools 記事へリンク
- 全記事 → ツールディレクトリ（/tools）へリンク

---

## SEOキーワード戦略

### 主要キーワード群

1. **ツール名 + レビュー/使い方:** 「Claude Code 使い方」「Cursor 比較」
2. **カテゴリ + おすすめ:** 「AIコーディングツール おすすめ」「AI画像生成 比較」
3. **課題 + 解決:** 「個人開発 収益化」「AIで稼ぐ方法」
4. **人名 + 戦略:** 「Pieter Levels 戦略」「ソロファウンダー 成功事例」
5. **ニュース + 解説:** 「OpenAI 最新 2026」「AI ニュース 今日」
